## v2 정정 (2026-05-18, 사용자 정정)

### 올리뉴 정체 정정
- 올리뉴는 **신규 브랜드 아님**
- **기존에 보유하던 건축물 유지보수 브랜드** — 이것을 **인테리어 리모델링 브랜드로 피벗(변경)**해서 새로 운영
- **넷폼알앤디의 브랜드 중 하나**로 자리매김할 예정

### 작업 우선순위 정정
사용자가 명시한 현재 우선순위:
1. **기존 올리뉴 홈페이지에서 단어·문장만 변경** (유지보수 → 인테리어 리모델링 카피 피벗)
2. **올리뉴 뉴스 보도기사 일단 발행**
3. (그 후) 브랜드 구체화 — 정체성·MVP·가설 수립

→ v1에 적었던 "MVP·톤·타겟·한 줄 소개·가설 수립"은 3단계 이후로 미룸.

### 첨부 자료
- 보도기사 본문: [[press_release_v0]] 에 보존

## v1 입력 (2026-05-18, 1차 — 일부 잘못 이해됨)

> ⚠️ v1은 "올리뉴 = 신규 브랜드"로 잘못 이해한 상태로 작성됨. v2가 정확.
> 단, 모회사·사업 방향·자산·차세대 기술 등 회사 컨텍스트 정보는 v1과 v2 모두 유효.

### 모회사 컨텍스트
- 모회사: 넷폼알앤디 (건축물 리노베이션 테크 기업)
- 회사 목표: 올해 브랜드 2개 런칭/MVP 검증/PMF 수익화, 1개월 내 브랜드 1개 런칭

### 사업 방향
- 실내 인테리어 + 건축물 리모델링 시장 동시 공략
- 기존 시공 중심 인테리어 시장에 **기술 기반 솔루션** 접목
- 공간의 **기능성·안전성·사용 경험** 통합 개선

### 보유 자산
- 특허기술 70여 종, 제품군 100여 종
- 기존 자산: 방수·도장·보수·외부 유지관리 → 실내 리모델링으로 확장 중
- 누적 270만 세대 공동주택 시장 기반
- 전국 250여 곳 시공사 네트워크
- 건축물 유지보수 기술, 소재 개발 역량, 시공 네트워크, 제조 인프라
- 연내 25년차 종합건설업체 인수합병 추진 중

### 차별화
- "단순 시공" X → **"기술+제품 결합형 솔루션"**

### 적용 분야
- 층간소음 저감 / 실내 안전성 향상 / 화재 위험 저감 / 공기질 개선
- 공용공간 기능 개선 / 공동주택 커뮤니티 시설 리모델링

### 차세대 기술 접목
- **AI**: 공간 이용 데이터(동선·체류시간·시설 활용도·관리 편의성) 분석 → 리모델링 방향 제안
- **모듈러 공법**: 공장 사전 제작 → 공기 단축 + 품질 편차 감소
- **로봇 시공**: 반복·정밀 시공, 위험 작업 보조
